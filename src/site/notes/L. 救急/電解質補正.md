# 低K血症
《治療原則》
最大投与速度：10～20mEq/hr
最大濃度：40mEq/L（末梢のみ）
最大投与量：100～150mEq/day

《点滴での基本セット》
生理食塩水 500mL＋KCL 20mEq
濃度：40mEq/L
投与速度：1hr以上（緊急時）/2hr以上（準緊急時）/4hr程度（通常時）

《内服での基本セット》
アスパラカリウム 300mg（1.8mEq） 3～9錠/分3
スローケー錠/ケーサプライ錠 600mg（8mEq）
