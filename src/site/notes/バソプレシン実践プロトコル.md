# 📋 敗血症性ショックにおけるバソプレシン使用プロトコル

**対象:** ノルアドレナリン(NA)不応性の血管拡張性ショック
**目的:** カテコラミン過量投与の回避 (Catecholamine Sparing) とショック離脱
**出典:** m3.com スペシャリストの視点（久志本成樹 医師）

---

## ✅ Phase 1: 導入判断 (Screening)

バソプレシンは「最後の手段」ではなく、**早期併用薬**です。

### 適応基準 (以下を全て満たす)
- [ ] 十分な輸液負荷が行われている (循環血液量が保たれている)。
- [ ] 平均動脈圧 (MAP) < 65 mmHg が持続している。
- [ ] ノルアドレナリン投与量が **0.25 µg/kg/min** (体重60kgで約15µg/min) に達している。
- [ ] ショック発症から **6時間以内** である。

### 慎重投与・除外
- [ ] **四肢虚血**、閉塞性動脈硬化症の既往がある。
- [ ] **腸管虚血**の疑いがある (激しい腹痛、乳酸値急上昇)。
- [ ] **活動性の冠動脈虚血** (ACS) がある (心拍出量低下のリスク)。

---

## 💉 Phase 2: 投与・調整 (Dosing)

**投与経路:** 安全性のため、原則として **中心静脈ルート (CV/PICC)** を使用する。

### 推奨される2つの投与法
どちらかの方法を選択します。

#### 1. 固定用量法 (Standard)
多くの施設での標準です。
* **0.03 Units/min** で開始し、そのまま**固定**する（体重換算は行わない）。

#### 2. 漸増法 (Cautious)
虚血リスクを最小限にする慎重な方法です。
1. **0.01 Units/min** から開始。
2. **20分ごと** に **0.005 Units/min** ずつ増量。
3. 目標血圧または上限用量に達したら固定。

> [!WARNING] 安全域と上限
> * **安全域:** 0.04 Units/min まで
> * **最大許容:** 0.06 Units/min (これを超えると虚血リスクが急増するため避ける)

---

## 🔢 付録: 流量換算表 (mL/hr)

シリンジポンプ使用時の早見表です。

### 【パターンA】標準希釈 (20単位/50mL)
* **組成:** ピトレシン 1A (20単位) + 生食 49mL = **合計 50mL**
* **濃度:** **0.4 単位/mL**

| 投与量 (Units/min) | 流量 (mL/hr) | 備考 |
| :--- | :--- | :--- |
| **0.01** | **1.5** | 開始用量 (慎重投与) |
| **0.02** | **3.0** | |
| **0.03** | **4.5** | **【基本用量】** (SSCガイドライン推奨) |
| **0.04** | **6.0** | 安全域の上限 |
| 0.06 | 9.0 | 最大許容量 (これ以上は避ける) |

---

## 👁️ Phase 3: モニタリング (Safety)

投与中は以下の項目を頻回に評価してください。

* **四肢虚血:** 指先・足先の色調、冷感、チアノーゼ、CRT延長
* **SpO2波形:** 波形の減弱・消失がないか
* **心機能:** 心拍出量の低下がないか (心エコー等)
* **腸管虚血:** 新たな腹部膨満、代謝性アシドーシスの進行

---

## 📉 Phase 4: 離脱・中止 (Weaning)

* ショック離脱時、ノルアドレナリンとどちらを先に中止するか明確な結論はない。
* 急激な中止は反動性低血圧を招く可能性があるため、**徐々に減量**することが望ましい (例: 1時間ごとに0.005 Units/minずつ)。
